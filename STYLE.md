# 블로그 후기 글 스타일 가이드

> 블로그 주인의 기존 맛집 후기 말투를 그대로 따라서 새 글을 작성한다.

## 말투 / 어미
- 친근하고 귀여운 존댓말: `~용`, `~어요`, `~답니다`, `~해용~~ㅎㅎ`
- 가끔 늘여쓰기: "다녀보고", "좋았어용", "나온답니다"
- 귀여운 감탄/마무리: `>ㅁ<`, `짠`, `빠잉`, `숭늉숭늉ㅎㅎ`, `ㅎㅎ`
- 이모지 적당히: 😊 ⏰ (남발하지 않기)

## 글 구조 (순서)
1. **인사** — "안녕하세요!! 다들 잘 지내셨나용~~"
2. **오늘 소개할 곳** — 장소 이름 크게
3. **위치 / 접근성 / 분위기** — 골목, 역 거리, 조용한지 등
4. **영업시간 / 실용정보** — ⏰ 이모지와 함께 정리
5. **메뉴 소개** — 가격대, 뭘 시켰는지
6. **음식/사진별 코멘트** — 사진 하나하나에 짧고 귀여운 설명
   - 예: "따듯해서 기분좋아요", "이게 포인트", "짠 이런식으로 나와요"
7. **총평** — 좋았던 점 (조용함, 맛, 분위기 등)
8. **추천 멘트** — "데이트 코스로도 추천드려요!>ㅁ<"
9. **마무리 인사** — "빠잉"
10. **해시태그** — #지역맛집 #장소명 #메뉴 #분위기맛집 등 6~9개

## SEO·조회수 규칙 ⭐ (2026 네이버 상위노출 대응)
네이버 C-Rank(전문성) + D.I.A+(진짜 경험) 알고리즘 대응. 글마다 아래를 지킨다.
1. **사진 6장 이상** 넣기 (직접 촬영 원본). 5장 이하면 후보에서 더 고른다.
2. **롱테일 키워드 제목**: 유명 상호 단독보다 "지역+메뉴+특징"으로 구체화.
   (예: "송쉐프" ❌ → "남부터미널 만원 점심특선 깐쇼새우정식" ✅)
   본문에도 핵심 키워드를 자연스럽게 2~3회.
3. **진정성/솔직함**: 단점·개인 감상·에피소드를 꼭 한두 개 넣는다 (승부처).
4. **AI 티 줄이기**: 매번 같은 틀 복붙 금지. 문장·순서·표현을 조금씩 다르게, 개인 경험 한 스푼.
5. **체류시간용 분량**: 정보(주소·가격·팁)+후기로 충분히 길게 (너무 짧게 X).
6. **동영상 리마인드**: 발행 안내 때 "15초+ 짧은 영상(음식 클로즈업 등) 1개 넣으면 노출 유리"라고 사용자에게 알려준다.
7. **꾸준함(참고)**: 한 주제(맛집)로 꾸준히 쌓여야 C-Rank 전문성 인정 (카테고리당 50개+ 목표).

## 메뉴·가격 규칙 ⭐ (항상 포함)
- **모든 글에 "📋 메뉴/가격" 섹션을 반드시 넣는다.** (맛집·카페·핫도그 등 종류 무관)
- 사용자가 준 가격이 있으면 그대로, 없으면 WebSearch로 확인해서 넣되 "가격은 변동될 수 있어요" 문구 첨부.
- 가격을 못 찾으면 사용자에게 물어봐서라도 넣는다. (그냥 빼지 말 것)

## 사진/내용 순서 규칙 ⭐ (2026-07 사용자 피드백)
- **상차림(먹는) 순서대로 배치**: 반찬 → 밑반찬 → 계란국/스프 같은 곁들이를 **앞쪽**에,
  대표 메뉴(메인 요리)는 **마지막**에 배치해서 클라이맥스로.
- 예: 반찬 → 계란국 → 사이드(만두/편육 등) → … → 메인 대표요리(맨 끝)

## 톤 규칙
- 솔직하지만 따뜻하게. 과장 광고 느낌 X
- 문장 짧게, 줄바꿈 많이 (블로그 가독성)
- 사진 설명은 한두 줄로 귀엽게
- 정보(위치·시간·가격)는 정확하게 챙기기
