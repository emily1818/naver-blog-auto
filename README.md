# 네이버 블로그 자동 글쓰기 🐾

사진만 넣으면 AI(Claude)가 사진을 분석해서 **귀여운 후기 + 설명글**을 써주는 시스템입니다.

## 폴더 구조
```
naver-blog-auto/
├── photos/      ← 여기에 사진을 넣으세요 (jpg, png, heic 등)
├── posts/       ← 완성된 글이 여기에 저장됩니다
└── new_post.sh  ← 실행 스크립트
```

## 사용법

### 1단계. 사진 넣기
`photos/` 폴더에 오늘 올릴 사진들을 넣습니다.

### 2단계. 글 만들기
Claude Code에서 이렇게 말하면 됩니다:

> **"네이버 블로그 글 만들어줘"**

그러면 Claude가:
1. `photos/` 안의 사진을 하나씩 봅니다
2. 사진 내용을 바탕으로 귀여운 후기글(제목 + 본문 + 해시태그)을 씁니다
3. `posts/` 폴더에 `YYYY-MM-DD-제목.md` 로 저장합니다

### 3단계. 네이버에 올리기
- **간단 방법**: 저장된 글을 복사해서 네이버 블로그 글쓰기에 붙여넣기
- **완전 자동(선택)**: "네이버에 자동으로 올려줘" 라고 하면 크롬 자동화로 글쓰기 화면에 자동 입력

## 스타일
- 톤: 귀엽고 친근한 후기체 (예: "오늘은요~ 🥰")
- 구성: 인사 → 사진별 설명 → 느낀 점 → 마무리 → 해시태그
- 이모지 적절히 사용

## 팁
- 사진 파일명에 장소/메뉴 등을 적어두면 더 정확한 글이 나옵니다
  (예: `성수동_감자탕.jpg`)
- 글을 다 쓴 뒤 `photos/`를 비우면 다음 글과 섞이지 않아요
